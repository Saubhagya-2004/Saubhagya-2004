# 🚀 Featured Projects

## 🍔 Food Hub – Multi Vendor Food Delivery Platform
**React.js, React Native, Node.js, Express.js, MongoDB, Redis, BullMQ, WebSocket, Razorpay, Firebase, Tailwind CSS**

Developed a production-grade multi-vendor food delivery platform with dedicated dashboards for Users, Delivery Partners, and Shop Owners. Implemented real-time order tracking and communication using WebSocket while integrating Razorpay for secure online payments. Engineered scalable backend workflows using Redis and BullMQ for asynchronous job processing, notifications, and order handling with optimized API performance.

---

## 💬 ChitChatto – Real-Time Group Chat Application
**React.js, WebSocket, Upstash Redis, Node.js, Express.js**

Built a scalable real-time group chat platform supporting concurrent multi-user communication using WebSocket architecture. Implemented Upstash Redis Pub/Sub for low-latency message broadcasting and efficient real-time synchronization across active chat rooms. Designed a modern responsive UI optimized for seamless user experience across devices.

---

## 🌐 DevTinder – Developer Networking Platform
**React.js, React Native, Node.js, Express.js, MongoDB, WebSocket, Nodemailer, Cloudinary**

Engineered a production-ready developer networking platform enabling developers to connect and collaborate through an intelligent swipe-based matchmaking system. Built secure authentication with Email OTP verification and implemented real-time one-to-one chat using WebSocket after successful matches. Developed a scalable React Native mobile application with optimized frontend architecture and cloud-based media management using Cloudinary.

---

## 🍽️ Dine Time – Smart Dinner Planning App
**React Native, Firebase, Tailwind CSS**

Developed a real-time dinner planning mobile application with reusable UI architecture and scalable state management. Integrated Firebase Authentication with secure Email/Password and OTP-based login workflows while enabling instant live synchronization using Firebase Realtime Database. Designed responsive mobile-first interfaces optimized for smooth cross-device user experience.
